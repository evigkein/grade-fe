




# Projects:
/project: projects/[projectName]/:
* domain: common features across the app which have specific logic 
* pages: all app component-pages (/portfolio, /smth, /etc)
* src: main routes definitions (not pages)
* public: assets images
* icons: svg icons

/shared/:
* features: Here we store features which are not single ltl ui-component
* ui/features: Here we store small ui-components-features
* ui/components: small dumb components
* ui/modals -- all modals components
